About This exhibit

— Theme —

This layout is themed on a fictional route from Sacramento, CA to San Diego, CA. There are 3 independant loops so up to 3 trains can be run at once. There is also a 2 track trolley line in the downtown area, and a narrow guage (On3) section in the logging town.

— Train Control —

The Cabrillo Southwest has both a DC and DCC control system. The DCC is controlled using NCE equipment and lets you conrole the lights, sounds, and speed of your locomotives. In the DC mode, a DC current is passed through the locomotive which only cotnrolles the speed. Many people still run using the DC because they have older locomotives thet they can't find DCC decoders for.
